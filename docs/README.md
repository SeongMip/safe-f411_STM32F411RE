# safe-f411 docs selection

이 폴더는 safe-f411 문서를 신입 임베디드 SW 포트폴리오 기준으로 다시 분류한 결과다.

## 폴더 설명
- `core/`: 먼저 보여줄 핵심 문서
- `optional/`: 있으면 좋은 보조 문서
- `archive/`: 현재 포트폴리오 기준으로는 과한 문서

## 기본 제출 추천 세트
- project overview
- architecture
- test strategy
- test procedure
- test report
- release note
- user guide
- developer guide

## 보조 제출 추천 세트
- module responsibility
- delivery package
- log rule
- judgement rule

## 제외 권장
- traceability matrix 전체 문서
