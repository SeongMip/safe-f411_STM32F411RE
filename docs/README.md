# safe-f411 Documentation

이 문서는 safe-f411 프로젝트의 문서 목록을 정리

## Core Documents

- `core/project_overview.md`
  프로젝트 목적과 전체 개요를 정리

- `core/architecture.md`
  firmware, RTOS, host tool의 구조와 역할 분리를 설명

- `core/test_strategy.md`
  테스트 설계 방향과 검증 관점을 정리

- `core/test_procedure.md`
  대표 TC의 시험 절차와 관찰 방법을 정리

- `core/test_report.md`
  실행 결과, 로그, PASS/FAIL 판정 결과를 정리

- `core/release_note.md`
  버전별 변경 사항과 검증 내용을 정리

## Supporting Documents

- `supporting/log_rule.md`
  UART 로그 형식과 태그 규칙을 정리

- `supporting/test_judgement_rule.md`
  PASS / FAIL / IN_REVIEW 판정 기준을 정리

- `supporting/user_guide.md`
  보드와 host tool 사용 흐름을 간단히 설명

- `supporting/developer_guide.md`
  개발자가 프로젝트를 빌드하고 수정할 때 참고할 내용을 정리

- `supporting/delivery_package.md`
  제출 또는 납품 패키지 예시 구조를 정리

- `supporting/traceability.md`
  요구사항, TC, 문서 간 연결 관계를 정리
